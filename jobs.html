Tech Jobs

This section of the Hypytenuse Webapp is where the major concepts come together. The application of code implemented was originally formed with the idea of learning how to code and using various methods to aggregate Tech industry related jobs.

The jobs section is catergorized by 4 fields: Company name, Job title, Location, and URL.

These fields were aggreated by using the Requests Library and Beautiful Soup for HTML. Additionally, we also utilized an API for Githubjobs.com. By doing so, we were faced with handling both structured and unstructured data and the integration of each.
